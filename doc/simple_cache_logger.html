Description

Configures Erlang Logger for the simple_cache application.

Logger configuration is read from the application environment under the logger key. The following options are supported:

* enabled - Enables or disables logging. * primary_level - Sets the Logger primary log level. * format - Selects the log output format (new or old).

If logging is disabled or the configuration is missing, the primary log level is set to none, effectively disabling log output. -------------------------------------------------------------------
